Rupee Game Ver0.3 Readme
はじめに

本プログラムは、Visual Studio 2022でC言語を使って作成した、exeファイルによって動作するゲームとなります。
本ゲームはWindows11で作成されたため、Windows11以外での環境で動作するかは不明です。

本ゲームの目的は、フィールドに隠されたお金を集めることです
通貨単位は1,5,20,100,300,-10です
なお、ゲーム内のマップには、（以下のように1から8の数字で表示されます）→通貨単位がそのまま表示されます。（Ver0.2で更新）
その意味は
101:爆弾（はずれ枠）
-10:-10ルピー（はずれ枠）
1:1ルピー
5:5ルピー
20:20ルピー
100:100ルピー
300:300ルピー
8:あなたはすでにそこを掘っています。
0:空白（エラーですので通常は出ません）

本ゲームは「マインスイーパー」のルールに類似しています
高いお金の周り(対象のお金を中心にして周囲8マス)ほど、はずれも多く埋まっています
逆に、低いお金の周りにははずれは少ないです
よって完全な運ゲーというわけではありません（最初の一択を除く）


難易度は
Easy
Normal
Hard
Lunatic(未実装)
Extra(未実装)

とわかれてます。下に行くほど難易度が高いです(Extraは例外)
高い難易度ほど、ハイリスクハイリターンです
まあ、失敗しても特にペナルティもないので気軽にやってみてください

※注意点
score.txtやAchievement.txtのファイル名や中身をいじると、ゲームが正しく動作しません。
スコアや実績をリセットしたい場合
スコア：score.txtの最上段（Name Scoreなど）以外の文字を消去してください
実績：Achievement.txtの中身を「完全に」削除してください（空白やスペース含む）
